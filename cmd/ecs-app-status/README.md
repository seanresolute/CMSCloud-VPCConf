# ecs-app-status

## Install

```
go install ./cmd/ecs-app-status
```

## Prerequisites for running
### Environment Variables
- CLOUDTAMER_BASE_URL
- CLOUDTAMER_ADMIN_GROUP_ID
- CLOUDTAMER_USERNAME
- CLOUDTAMER_PASSWORD
- CLOUDTAMER_IDMS_ID

### artifact-db
Install from https://github.cms.gov/superbrilliant/artifact-db
